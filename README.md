# arethafw
A PHP Library for common Web applications


- - - -

Some of the main features includes:


* Multiple database access (PostgreSQL, MySQL)
* Server side form validation
* Session handling
* User access handling

- - - -

## Initialization

```php
'include 'arethafw/Aretha.php';
```
